Intro and Contents

Its intuitive, interactive interface means it can be used by any individual having a pionnering soul.

How To Create Your Own Plot

You can interact with the plot located in the right panel by changing the input controls on the left. from top to bottom:

X: select the variable you want to display on the horizontal axis

Y: Select the variable you want to display on the vertical axis

Color: select the variable you want to use for colouring the points within the plotting area

Smooth: tick this checkbox if you want to add a smoothed conditional mean

Facet Row: select the variable you want to use to split up your data and plot the subsets (facets) of data together.
